All notable changes to SupplyLines will be documented in this file.

## [Unreleased]

## [1.0.0-alpha.1] - 2025-12-29

### Added
- Stock Keeper job and building for MineColonies integration
- Create mod logistics network integration
- Automatic item delivery from Create vaults to colony buildings
- Request system resolvers for various item types (stacks, tags, tools, food)
- Mixin support for AbstractBuilding integration
- CreateVaultPlacementHandler for Structurize compatibility
- Semantic versioning with pre-release and build metadata support
- ModVersion utility for runtime version access

### Technical
- Implemented request handling pipeline with delivery verification
- Added support for rack-based inventory management
- Network integration for Create belts, chutes, and smart observers
