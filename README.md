# SupplyLines

A Minecraft Forge mod that integrates MineColonies with Create mod, enabling colonists to request items from Create's logistics networks. Supply your colony using belts, chutes, and smart observers.

## Requirements

- Minecraft **1.20.1**
- Java **17**
- Forge **47.x**
- [MineColonies](https://www.curseforge.com/minecraft/mc-mods/minecolonies) 1.1.1069+
- [Create](https://www.curseforge.com/minecraft/mc-mods/create) 6.0.6+
- [Structurize](https://www.curseforge.com/minecraft/mc-mods/structurize)

## Building from Source

```bash
./gradlew build
```

Output JAR will be in `build/libs/`.

## Development

```bash
./gradlew runClient
```
